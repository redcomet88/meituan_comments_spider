# 美团外卖店铺信息与评论抓取脚本使用指南

## 一、工具简介

本脚本用于抓取美团外卖的店铺信息和用户评论数据，需要配合 **Reqable（小黄鸟）** 抓包工具使用。

### 抓取内容

| 数据类型 | 说明                                                         |
| -------- | ------------------------------------------------------------ |
| 店铺信息 | 名称、ID、评分、月销量、地址、电话、起送价、配送费、配送时间、配送服务、商家公告等 |
| 用户评论 | 用户名、评分、评论内容、评论时间、订单时间、配送时间、点赞食物、商家回复、评论图片 |

## 二、环境准备

### 1. 下载安装 Reqable

访问官网下载：https://reqable.com/zh-CN/

支持 Windows、macOS、Linux、Android、iOS 等平台。

### 2. 配置脚本路径

编辑脚本文件 `美团外卖[1.0]店铺信息+评价同步抓取.py`，修改第7行的数据保存路径：

```python
BASE_DIR = r'/Volumes/tesla/dev/flask/f082-meituan-comments/data'
```

将路径改为你本地的实际路径。

## 三、Reqable 配置步骤

### 步骤 1：添加脚本

1. 打开 Reqable，进入 **脚本** 功能页面
2. 点击 **添加脚本** 或新建脚本
3. 将 `美团外卖[1.0]店铺信息+评价同步抓取.py` 的完整代码复制粘贴进去
4. 保存脚本

### 步骤 2：配置匹配规则

脚本会自动匹配以下美团外卖接口：

| 接口路径                                                     | 功能         |
| ------------------------------------------------------------ | ------------ |
| `https://wx.waimai.meituan.com/weapp/v1/poi/info`            | 店铺基本信息 |
| `https://wx.waimai.meituan.com/weapp/v2/search/v9/poi`       | 搜索店铺列表 |
| `https://wx.waimai.meituan.com/weapp/v1/poi/food`            | 店铺菜单详情 |
| `https://wx.waimai.meituan.com/weapp/shop/v1/poi/productlist` | 店铺商品列表 |
| `https://wx.waimai.meituan.com/weapp/v1/comment/poi`         | 用户评论     |

### 步骤 3：启用脚本

1. 在脚本列表中启用刚添加的脚本
2. 确保脚本类型设置为 **响应脚本（Response Script）**
3. 开启 Reqable 的抓包功能

## 四、抓取操作流程

### 方式一：手机端抓取（推荐）

1. **手机配置代理**
   - 确保手机和电脑在同一局域网
   - 手机 WiFi 设置代理，指向电脑 IP 和 Reqable 监听端口（默认 9000）

2. **安装证书**
   - 在 Reqable 中导出 CA 证书
   - 手机安装证书并信任

3. **开始抓取**
   - 打开美团 APP，进入外卖频道
   - 搜索或浏览目标店铺
   - 点击进入店铺详情页
   - 切换到 **评价** 标签页
   - 滑动浏览更多评论（脚本会自动抓取）

### 方式二：模拟器/微信小程序抓取

1. 使用安卓模拟器或手机微信打开美团外卖小程序
2. 配置代理指向 Reqable
3. 操作流程同上

## 七、注意事项

1. **合规使用**：本脚本仅供学习研究使用，请勿用于商业用途或大规模爬取
2. **频率控制**：避免频繁请求，建议自然浏览
3. **数据完整**：需要进入店铺详情页并点击评论页才能抓取评论数据
4. **路径配置**：首次使用务必修改脚本中的保存路径
5. **编码格式**：CSV 文件使用 UTF-8-BOM 编码，Excel 可直接打开

## 八、常见问题

### Q1: 抓不到数据？

- 检查 Reqable 抓包是否开启
- 确认脚本已启用且匹配的 URL 正确
- 检查证书是否正确安装和信任

### Q2: CSV 文件乱码？

- 使用 Excel 打开时选择 UTF-8 编码
- 或使用支持 UTF-8 的编辑器（如 VS Code）打开

### Q3: 店铺信息不完整？

- 部分信息需要从不同接口合并
- 建议先搜索店铺，再进入详情页，最后查看评论页

## 九、技术说明

脚本通过拦截美团外卖小程序/API 的 HTTP 响应，解析 JSON 数据并保存到本地文件。主要处理流程：

```
HTTP 响应 → 解析 JSON → 提取数据 → 写入文件
```

核心函数：

- `onResponse()` - 响应拦截入口
- `extract_base_info()` - 提取店铺基础信息
- `ensure_shop_csv_header()` - 创建/更新 CSV 文件头
